# Publishing Checklist

Use this checklist before sharing Textbook or any generated course app publicly.

## Textbook Template

- The public `textbook` repository contains template code only, not copied course content.
- No `.env.local`, `.vercel`, `.next`, `node_modules`, database URLs, API keys, cookies, or personal deployment settings are committed.
- The template code remains under `LICENSE.md` using PolyForm Noncommercial License 1.0.0.
- `CONTENT_POLICY.md` is linked from the README.

## Generated Course Apps

- Work happens in the user's own fork, not in the shared template repository.
- The source repository has a detected license or written permission.
- If no source license is detected, do not publish the generated course.
- `ACKNOWLEDGEMENTS.md` names the source repository, source commit, source license, and upstream authors/contributors.
- The app visibly states that the content was adapted into a study interface.
- The app does not imply endorsement by upstream authors.
- The generated app is private unless the source license clearly permits public redistribution and derivative works.
- If the source license is noncommercial, the generated app is not monetized, sold, or used in a paid product/service.
- If the source license is share-alike or copyleft, the generated app follows those obligations before publication.

## Practical Default

When in doubt, keep the generated course private, use your own notes/content, or ask the rights holder for permission.
